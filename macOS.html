An operating system developed and marketed by Apple Computer since 1999 under the name Mac OS X Server 1.0. Then in 2001 for home appliances as Mac OS X 10.0. It works on PowerPC processors and more recently Intel processors, built on the BSD system, which is one of the Unix systems in addition to the Mac core, which is a development for Mac Os that began its appearance in 1984. The first release of the Mac OS X was in March 2001. The seventh version (10.7) of Mac OS X, which was released in 2011, also known as "Mac OS Sierra", is the first computer operating system for Apple Which fully supports the Arabic language.
